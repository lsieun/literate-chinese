## 五笔字根识别

五笔字根识别码：拆完字根还打不出来加识别码键,同理复杂的字取前三个字根加最后一笔

- （1）、末笔为“一”
  - 左右型的汉字，末笔字型识别码为：G。
  - 上下型的汉字，末笔字型识别码为F。
  - 杂合型的汉字，末笔字型识别码为D。
- （2）、末笔为“丨”
  - 左右型的汉字，末笔字型识别码为H。
  - 上下型的汉字，末笔字型识别码为J。
  - 杂合型的汉字，末笔字型识别码为K。
- （3）、末笔为“丿”（左右：T、上下:R、杂合：E）
- （4）、末笔为“丶”（左右：Y、上下:U、杂合：I)
- （5）、末笔为“乙”（左右：N、上下：B、杂合：V)
